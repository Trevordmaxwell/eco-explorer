# Season Rhythm And Stop Points Phase

## Summary

Lane 1's next later wave after the current long-push season-two work should focus on pacing and confidence rather than new systems.

The next best gains are:

1. clearer stop-point rhythm across route launch, return, and filing moments
2. one compact follow-on that makes the season feel easier to pause and resume

## Why Now

Once the current travel and shell waves land, the player should be able to feel where a good stopping point is. That is a high-leverage polish seam for longer sessions and return sessions alike.

## Recommended Order

1. Prepare the season-rhythm handoff.
2. Implement the stop-point rhythm pass through current wrap, routes, and expedition seams.
3. Review whether the season is easier to pause and resume.
4. Prepare one compact route-resume follow-on.
5. Implement the route-resume pass inside existing station and journal seams.
6. Review whether the game now feels easier to re-enter.

## Guardrails

- Do not add a separate recap screen.
- Do not add another planner or task log.
- Reuse existing strip, board, wrap, expedition, journal, and map seams.

## Intended Result

After this phase, the season should feel easier to stop, resume, and carry in memory without a larger management shell.
