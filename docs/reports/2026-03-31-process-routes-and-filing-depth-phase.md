# Process Routes And Filing Depth Phase

## Summary

Lane 4's next gameplay-loop wave should deepen outings in two ways:

1. add one process-backed Route v2 outing that uses the live world-state seam as gentle opportunity rather than punishment
2. add one slightly richer notebook filing step that still fits the current compact station shell

## Why Now

Route v2 now supports landmark, evidence, ordered, and transect-like play. The next high-leverage move is to let one outing ask the player to notice an ecological process or condition, then make the return filing feel a little more like synthesis instead of just confirmation.

## Recommended Order

1. Prepare the first process-backed Route v2 conversion.
2. Implement the process-backed outing.
3. Review it for clarity, softness, and truthful use of world-state.
4. Prepare one compact filing-depth follow-on.
5. Implement the filing-depth follow-on.
6. Review whether the outing loop now feels more game-like without becoming quiz-like.

## Guardrails

- Use world-state as soft opportunity, not hard punishment.
- Do not add timers, failure states, or a larger station shell.
- Keep the filing step notebook-first and one-screen.
- Avoid broad new route ledgers or inventory systems.

## Intended Result

After this phase, one live outing should feel more like “notice a process in place and bring back a field case,” and the filing return should feel a little more authored and satisfying.
