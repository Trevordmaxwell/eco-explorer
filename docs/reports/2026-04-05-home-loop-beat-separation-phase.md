# Home-Loop Beat Separation Phase

## Summary

The home loop now needs clearer sequencing, not more surface area.

Lane 4 should make nursery and home-loop payoffs read as distinct beats:

- plant state
- ecological reflection
- utility or reward

Those should not all fight for the same card at the same time.

## Goals

- separate nursery beats so the player reads one idea at a time
- keep home-loop payoff satisfying without adding another planner layer
- reinforce that the home loop changes because of what the player learned in the field

## Wave Shape

### Step 2

- implement one beat-separation pass for nursery interaction state

### Step 3

- review whether the home loop now reads as cleaner, not thinner

### Step 4

- prepare one reward-surfacing follow-on handoff

### Step 5

- implement one compact reward or utility surfacing seam outside the crowded nursery card

### Step 6

## Guardrails

- no new dashboard, inventory, or larger home-base shell
- avoid putting every beat back into a single text card
- keep the station and nursery calm, compact, and readable
