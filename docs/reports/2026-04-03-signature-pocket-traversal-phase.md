# Signature Pocket Traversal Phase

## Summary

Lane 3 should now give the world more unforgettable spatial identity.

The review's most useful spatial call is simple:

- every biome should eventually have one unforgettable signature pocket
- `Coastal Scrub` especially needs a stronger destination-like identity
- `Treeline` and `Tundra` should feel more like exposure and threshold spaces, not just content variants

## Goals

- add one memorable `Coastal Scrub` signature pocket
- add one stronger high-country threshold pocket
- keep both cozy, readable, and recoverable

- no traversal HUD growth
