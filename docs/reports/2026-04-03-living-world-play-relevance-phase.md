# Living-World Play Relevance Phase

## Summary

Lane 4 should make living-world systems matter to play instead of reading as flavor only.

The strongest product recommendation in the review is:

- use day-part, weather, phenology, and process moments as soft opportunity
- let routes feel different because of world state
- make support choices meaningfully change how an outing feels, or keep them tiny

## Goals

- add one route-aware living-world play pass
- make one support-choice layer meaningfully different in practice
- keep all of it inside the current Route v2 and note/support seams

## Guardrails

- no new shell, quest log, or inventory layer
- no hard gating or punishment timers
- prefer soft opportunity, clue emphasis, and replay value
- if a support still cannot create a felt difference, simplify it instead of bloating it
