# Forest Vertical Expedition Phase

## Summary

Lane 3's second long-push wave should start making the strongest forest vertical spaces feel more chapter-ready.

The next best shape is:

1. add one calmer forest vertical expedition seam across the giant-tree and cave family
2. add one compact old-growth or under-root sub-space that supports that outing feel

## Why Now

Forest now has the richest vertical language in the game. The next gain is not just another ledge or chamber. It is making those spaces feel more like one memorable outing path.

## Recommended Order

1. Prepare the forest vertical expedition seam.
2. Implement that seam.
3. Review whether it feels chapter-grade and readable.
4. Prepare one compact support sub-space.
5. Implement the support sub-space.
6. Review whether the forest vertical family now feels more outing-ready.

## Guardrails

- Keep the spaces readable and recoverable.
- Avoid combat, hazards, or a denser cue language.
- Reuse the current vertical family rather than starting over.
- Keep the additions compact.

## Intended Result

After this phase, forest vertical play should feel closer to a real destination outing, not just a set of strong connected proofs.
