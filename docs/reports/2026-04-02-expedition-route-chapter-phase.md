# Expedition Route Chapter Phase

## Summary

Lane 4's later long-push wave should push one route family fully into expedition-grade chapter territory while keeping the cozy notebook-first tone.

The next best gains are:

1. one compact expedition-route chapter conversion
2. one follow-on that makes the return page feel more like the close of a chapter

## Why Now

By this point the Route v2 loop should be mature enough that one family can carry a fuller chapter arc without needing a new shell or a genre pivot. That is one of the highest-value gameplay moves available.

## Recommended Order

1. Prepare the expedition-route chapter handoff.
2. Implement the chapter-grade route conversion.
3. Review whether the outing now feels like a fuller chapter.
4. Prepare one compact return-page follow-on.
5. Implement the return-page pass on top of the live notebook return seam.
6. Review whether the chapter close feels satisfying and compact.

## Guardrails

- Do not add combat, failure meters, or a bigger quest shell.
- Reuse current Route v2, support, and notebook-return seams.
- Keep the outing calm, authored, and place-specific.

## Intended Result

After this phase, at least one route family should feel like a true expedition-grade chapter while still fitting the cozy naturalist direction.
