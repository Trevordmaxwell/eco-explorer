# 2026-03-29 Next High-Leverage Phase

## Recommendation

The biggest next leverage is not another isolated feature.

It is one short authored gameplay arc that makes the current systems feel like one game.

Recommended order:

1. first guided field-season loop
2. content fuel pack for that loop
3. second traversal proof in another biome
4. second-tier field-station upgrade layer
5. later ecology process moments and richer seasonal revisit change

## Why This Is The Right Shift

The project now already has:

- multi-biome travel
- a stable corridor model
- notebook requests
- a field station and first upgrade
- sketchbook, sound, and close-look layers
- living-world state through day-part, weather, phenology, prompts, and partner cues

What it still lacks is a compact authored arc that tells the player:

- what to do first
- why to go somewhere specific
- how the systems connect
- what the first satisfying “I made progress” loop actually is

That makes the highest-value next move a cohesion pass, not a feature buffet.

## Phase 1. First Guided Field Season Loop

Build a short fresh-save arc, roughly `15-20` minutes, that introduces the live systems in a good order.

Suggested beats:

1. start near the field station with one clear notebook request
2. send the player through one strong exploration beat, likely the forest hollow route
3. require one meaningful discovery or comparison
4. route the player back through the station for one small reward or acknowledgment
5. point toward the next habitat without forcing a giant quest log

This is the highest leverage because it turns travel, traversal, requests, journal, and station progress into one loop players can actually feel.

## Phase 2. Content Fuel Pack

Once that loop skeleton exists, add more authored fuel where the systems are now thin.

Best scope:

- at least one new strong discovery per biome family touched by the early loop
- one or two new ecosystem-note or request-support pairs
- one more comparison-worthy shared species pair
- one more sketchbook-worthy landmark or organism per high-traffic biome

This matters because the current systems are now broader than the authored content feeding them.

## Phase 3. Spread Traversal To A Second Biome

The forest proof is a good start, but one traversal-heavy biome is not enough to make movement feel like a core pillar.

Best next candidate:

- `coastal-scrub` or `treeline`

Why:

- `coastal-scrub` can deepen the world-continuity lane with sheltered dune cuts and shrub cover
- `treeline` can teach exposure and shelter in a very ecology-rich way

## Phase 4. Field Station Tier 2

After the first loop and second traversal proof are fun, add one more gentle station layer:

- one second upgrade
- one utility unlock
- one clearer long-term field-season goal

Keep the framing observational and science-safe.

## Phase 5. Ecology Process Moments

Only after the arc, content, and second traversal proof are holding should the next living-world expansion wake up again.

Best later direction:

- visible ecology process moments tied to weather and phenology

Examples:

- dunegrass trapping more sand
- fog-softened coast mornings
- brief flowering rushes
- berry or seed timing affecting what feels active

That will land much harder once the player already has reasons to revisit places.

## Queue Outcome

- queue one new main implementation lane for the first guided field-season loop
- queue one scout synthesis pass to beat-map that loop in parallel
- keep later content, traversal, and station expansions parked behind the first loop review
