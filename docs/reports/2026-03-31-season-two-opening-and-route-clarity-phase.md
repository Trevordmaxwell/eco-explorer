# Season-Two Opening And Route Clarity Phase

## Summary

Lane 1's next systems wave should make the game feel ready to continue after the first season close instead of quietly stalling at the archive screen.

The best next step is a compact two-part pass:

1. open the second season through the existing station and map seams
2. keep the active outing readable across the station, journal, and world map

This keeps progression moving without opening a larger quest shell or another station page.

## Why Now

The current build now has enough finished structure that the next risk is not missing systems. It is loss of momentum.

The logged season and expedition footer already hint at `High Pass`, but the game still needs one calmer, clearer bridge from "season complete" into "here is the next field season." At the same time, route discoverability is still fragile enough that lane 1 should reinforce active-outing clarity inside the surfaces the player already uses.

## Recommended Order

1. Prepare a compact second-season opening and `High Pass` launch handoff.
2. Implement the second-season opening pass inside the current station and map flow.
3. Review whether that opening feels inviting, singular, and calm.
4. Prepare one active-route discoverability follow-on.
5. Implement compact active-outing surfacing across station, journal, and map seams.
6. Review whether the route is now consistently easy to relocate after breaks or returns.

## Guardrails

- Do not add a new top-level station page.
- Do not add a larger quest log or navigation HUD.
- Keep one clear next outing rather than multiple simultaneous route pushes.
- Reuse the current strip, footer, board, and journal seams wherever possible.

## Intended Result

After this phase:

- the game should feel like it has a clear second season waiting
- the player should have one obvious next outing
- the station, journal, and world map should reinforce the same active route instead of leaving it easy to lose
