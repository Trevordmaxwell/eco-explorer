# Light-Band Route Approach Phase

## Summary

Lane 3 should not jump straight back into another signature destination.

The next useful lane-3 support pass is a tiny approach or recoverability seam in a lighter-weight route band so a route feels more tactile in motion without adding another landmark.

Priority order:

- `Tundra` first
- `Coastal Scrub` only if the chosen band stays away from the already-dense bluff lookout family

## Goals

- support one existing outing with a clearer in-world approach or read
- spend geometry budget on feel, not destination count
- keep the pass cozy, recoverable, and low-density

## Guardrails

- no new signature pocket or lookout family
- no more beach opener or treeline density work right now
- keep the pass small enough that it reads as support for route feel, not a new level
