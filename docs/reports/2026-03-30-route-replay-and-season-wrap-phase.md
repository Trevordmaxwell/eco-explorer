# Route Replay And Season Wrap Phase

## Summary

The queue is clear after the three-route station pass, so the next highest-leverage move is not a fourth live route. The current station shell is already at its safe density for `256x160`.

The best next phase is:

1. make the three live routes more replayable through route-aware revisit variants
2. add one compact season-wrap or next-outing teaser so station stops feel like chapter endings
3. mirror the inland continuity work on the coastal side with quiet authored corridor beats

This keeps the project in the same game-feel lane:

- more cohesive
- more replayable
- more chapter-like
- no heavier dashboard

## Why This Phase

- Three routes are enough to feel like a field season.
- A fourth route would put pressure on the current station shell before the current routes feel fully lived-in.
- The game already has day-part, weather, phenology, process moments, the nursery, and route progression. The next leverage is using those systems together better.
- The inland side already has quiet corridor follow-through; the coast now needs the same authored continuity.

## Direction

### 1. Route-aware revisit variants

Use the current living-world seams to make repeat outings feel meaningfully different without building a bigger simulation system.

Good variant hooks:

- day-part emphasis
- calm weather emphasis
- phenology emphasis
- process-moment emphasis
- route-specific notebook or nursery clue windows

Guardrails:

- no schedule calendar
- no rotating chores
- no route bloat

### 2. Compact season wrap

The field station should give a slightly stronger sense of closure after a route or outing:

- one tiny season note
- one quiet “good stopping point” recap
- one soft pointer toward the next outing

This should feel like a notebook margin, not a score screen.

### 3. Coastal corridor authored beats

The inland seams already have light authored follow-through. The coastal side should get the same treatment so the whole five-biome chain feels equally authored.

Best targets:

- `beach <-> coastal-scrub`
- `coastal-scrub <-> forest`

Keep it quiet:

- carriers first
- one small prompt window only where it helps
- no new corridor UI layer

## Recommended Queue Order

1. `main-79`: first route-aware revisit variant pass
2. `critic-57`: review replay subtlety and chapter feel
3. `main-80`: compact season-wrap and next-outing teaser
4. `critic-58`: review station closure tone
5. `main-81`: coastal corridor-authored beats
6. `critic-59`: review coastal continuity

Parallel scout prep:

- `scout-48`: revisit variant matrix
- `scout-49`: season-wrap copy and station flow
- `scout-50`: coastal corridor beat matrix

## What Not To Do Yet

- do not add a fourth live route yet
- do not add a heavier seasonal calendar
- do not add a market, shop, or bigger economy
- do not broaden the nursery further in this phase
- do not add another movement-upgrade ladder

## Success Signal

At the end of this phase, the player should feel that revisiting the current world is worthwhile, that the field station gives a satisfying place to stop, and that the whole biome chain feels more intentionally authored from coast to tundra.
