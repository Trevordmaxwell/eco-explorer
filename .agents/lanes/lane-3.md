# Lane 3: Retired Into Lane 2

Lane 3 is not an active lane in the current director-plus-two-lanes model.

Former lane 3 responsibilities now route through:

- `lane-2`: world, content, spatial richness, vertical traversal, cave/tree/tundra proof, and sub-ecosystem depth

Historical reports, packets, and done queue items may still mention lane 3. Treat those labels as archive context unless the director explicitly reactivates lane 3.

If a fresh agent is assigned lane 3, it should stop and ask whether the work should be routed to lane 2.
